Mini Pipeline — browser smoke (manual)

Serve the parent folder (e.g. python3 -m http.server 8080) and open index.html. No npm dependencies for this checklist.

  • Main controls: add Process/File, link, delete selection, load sample, load JSON, Save/Export — all above the graph.
  • Graph controls: only Reset zoom (fit graph + links + margin in view) and Cleanup next to the canvas.
  • Export downloads mini-pipeline.json; re-load it via Load JSON.
  • Cleanup uses a layered layout from links (simple chains read Process→File→Process) without changing labels or link count; export includes coordinates and reload restores layout.
